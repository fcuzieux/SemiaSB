🎵 Page de Test Audio pour SemiaSB

📋 Instructions

Cette page permet de tester la capture audio de SemiaSB :

  1. Cliquez sur "Jouer un son test" ci-dessous
  2. Ouvrez capture.html dans un autre onglet
  3. Cochez "Capturer l'audio"
  4. Cliquez sur "Démarrer"
  5. IMPORTANT : Cochez "Partager l'audio" dans la boîte de dialogue
  6. Sélectionnez cet onglet
  7. Vérifiez que le message indique "vidéo + audio"
